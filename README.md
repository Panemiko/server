# Surredstone server

## Download
- Clique em `Code` > `Download ZIP`

## Configurando
- Extraia o arquivo `zip` em qualquer pasta
- Adicione os mundos `world`, `world_nether` e `world_the_end`
- Adicione os arquivos de salvamento dos plugins
- Adicione os arquivos do servidor relacionados Ã  players (opcional)
	- `/banned-ips.json`
	- `/banned-players.json`
	- `/ops.json`
	- `/permissions.yml`
	- `/usercache.json`
	- `/whitelist.json`

## Rodando
- Instale Java 17 ou superior
- Execute o arquivo `start.bat`
